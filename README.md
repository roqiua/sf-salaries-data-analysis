# SF Salaries Data Analysis

A data analysis project on the **San Francisco Salaries** dataset using **Python**, **Pandas**, **NumPy**, and **Matplotlib**.

## 📌 Project Overview

This project focuses on analyzing the SF Salaries dataset by cleaning the data, handling missing values, exploring salary components, and creating visualizations to extract meaningful insights.

> **Note:** The Exploratory Data Analysis (EDA) for this dataset was completed in a separate repository.

## 🛠️ Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## 📊 Project Tasks

* Data cleaning and preprocessing
* Handling missing values
* Data validation
* Salary analysis
* Job title analysis
* Year-based salary comparison
* Data visualization
* Business insights

## 📂 Dataset

* **SF Salaries Dataset**
